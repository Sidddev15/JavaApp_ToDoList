Project Title: Java To-Do List Application

Description:

This Java To-Do List Application is a simple yet functional console-based program that allows users to manage their to-do tasks efficiently. It provides basic functionality for adding, removing, and viewing to-do list items, making it a valuable tool for personal task management.

Features:

Add Item: Users can add new tasks to their to-do list by providing a task description. The application ensures that the entire task description, including spaces, is captured.

Remove Item: The program allows users to remove tasks by specifying the exact task description. It checks for the existence of the task and provides feedback on successful removal.

View Items: Users can view their entire to-do list, which includes all added tasks. The application displays the complete task description.

Menu-Driven Interface: The application features a user-friendly menu-driven interface that guides users through these actions, making it easy to manage their tasks.

Usage:

Clone the repository to your local machine.
Compile and run the Java program using your favorite IDE or terminal.
Follow the on-screen instructions to add, remove, and view your to-do list items.
